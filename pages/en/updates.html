Updates
Before 1.5
I am sorry that there are no records before version 1.5.
1.6
A palette is added for your customing BG color or font color. There will be a one-time popup of updates when you click the extension icon.
1.7
A Rate It item is added for going to the extension's Rate It page.
1.8
Delete the one-time popup added at version 1.6, instead of a "Updates" page.
2.0.0
A new context menu "Disable at this site" which is suggested by user Lake Peak is added and I like it.
The item can disable the extension at one site, and the rule is get the domain name of the site,
and then disable the extension on all the pages of the site.
A new privilege "Access your tabs and browsing activity" is required.
2.1.0
The Background Color Options is default folded, and you can unfold it by click it.
2.1.2
A new menu "Disable extension" is added to the context menu that you can disable or
enable the extension by click the menu.
2.3.0
Now all the change to any options will immediately response to all the web pages.
3.0.0
A new mode "Night Mode" is add to the extension.
